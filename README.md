
SSAA – Stage1 – Step 4A.5
========================
Cabinet module – Consolidation release

This package represents step 4A.5 of SSAA Stage1.
Purpose:
- Consolidate cabinet pipeline from 4A.4
- Freeze a stable baseline before expansion (4A.6)
- No functional extensions introduced

Status: STABLE (intermediate)


## Ejecución (recomendado)

- En desarrollo (desde la raíz del proyecto):

  python -m ssaa

- Instalado como paquete (opcional):

  pip install .
  ssaa

> Nota: ya no se modifica sys.path en runtime. Para ejecución desde accesos directos, se recomienda usar el script/ejecutable instalado (o PyInstaller).
